console.log(): This method is used to output data to the browser's console. It is primarily used for debugging purposes to log information about the state of the program. For example console.log("Hello, world!"); document.write(): This method writes HTML expressions or JavaScript code to a document. It is not commonly used in modern web development because it can overwrite the existing document content and cause performance issues. For example: document.write("Hello, world!"); innerHTML property: This property sets or returns the HTML content (inner HTML) of an element. It allows you to dynamically update the content of HTML elements on the page. For example: document.getElementById("myElement").innerHTML = "Hello, world!"; alert(): This method displays an alert dialog box with the specified message and an OK button. It is often used to provide feedback to the user or to display important information. For example: alert("Hello, world!"); prompt(): This method displays a dialog box that prompts the user for input. It is commonly used to get data from the user. For example: var name = prompt("Enter your name:"); confirm(): This method displays a confirmation dialog box with a message and OK and Cancel buttons. It is often used to ask the user for confirmation before performing an action. For example: var result = confirm("Are you sure you want to delete this item?");
